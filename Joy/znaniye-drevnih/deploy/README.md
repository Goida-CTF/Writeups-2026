# Знание Древних

Таск деплоится из Docker Compose, все environment vars проставлены за исключением `FLAG` - его нужно заменить на значение из [`../solution/flag.txt`](../solution/flag.txt).

**Важно:** env var `FLAG` должен передаваться **только** в контейнер `backend` - иначе это может привести к случайному солву из контейнера `piston` (маловероятно, но всё же).
